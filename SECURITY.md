# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| latest  | :white_check_mark: |
| < 0.1.0 | :x:                |

## Commit Signing

All commits and release tags must be signed with an SSH or GPG key associated
with an identifiable human contributor. Unsigned commits will not be merged. See
[CONTRIBUTING.md](CONTRIBUTING.md#commit-signing) for setup instructions.

## Reporting a Vulnerability

Please report security issues to **john@opsdev.nz**. Do not open a public issue.

We aim to acknowledge reports within 72 hours and provide a fix or mitigation
within 90 days.
